# CB Add Ons

CB Add Ons are small packages of files that can be pasted into a CB-CSV project to add additional functionality. 
These are typically libraries and includes to support additional object formats.

The "README" of each Add On will have instructions about how to add it to your CB-CSV project.

Current Add Ons:

- [o3dv_cb-add-on](https://github.com/CollectionBuilder/o3dv_cb-add-on) - provides library and template files to add an [Online3DViewer](https://github.com/kovacsv/Online3DViewer) option into a CollectionBuilder-CSV project. It uses the o3dv library to allow embedding an interactive 3d model into pages in your collection project.
- [rti-viewer_cb-add-on](https://github.com/CollectionBuilder/rti-viewer_cb-add-on) - provides library and template files to add an RTI viewer option into a CollectionBuilder-CSV project. Reflectance Transformation Imaging (RTI) is an imaging technique used to reveal surface characteristics of 3d objects. RTI viewers allow users to change lighting angles and characteristics to explore the surface features and details. This add-on uses Relight and OpenLIME to allow displaying RTI items directly in your CollectionBuilder digital collection.
