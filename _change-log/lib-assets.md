---
title: Library Assets Location
release: 2025-09-29
---

The option `lib-assets` was added to "_config.yml" which allows you to set an external location for the library assets used by your project. 
This means if you have multiple CB projects, you can host your library assets (Bootstrap, Leaflet, DataTables, etc) in a central location, rather than having repetitive copies on your server.
Most users will not use this option, but it is helpful for organizations to build many CB collections.
