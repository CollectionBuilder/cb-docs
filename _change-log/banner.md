---
title: Banner and Nav Updates
release: 2026-02-18
---

The banner and nav templates have been updated to allow more flexibility and improve accessibility:

- Updated Collection Banner - "collection-banner.html" include was updated to be easily reuseable on any page (with front matter options) and to mirror the style of the About page featured image.
- New Side Nav Option - use the new "collection-side-nav.html" include in the default layout to add a basic navbar with a navigation menu that opens from the side. This significantly changes up the theme without requiring any configuration changes.
- Navbar Theming Update - the navbar feature was updated to use Bootstrap's new `data-bs-theme` option for theming. The Lunr search box was simplified to match the new theme options.

Look at the comments at the top of the include files for information about using these options.
